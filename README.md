# PDF Q&A with Groq LLM

An interactive Python application that allows users to **upload any PDF document and ask questions** about its content. This project uses **LangChain**, **vector embeddings**, and **Groqâ€™s LLM** to provide **context-aware, human-like answers**.

---
## Usage
- Make sure you have a valid Groq API key set in your environment variables.
- Works best with well-formatted PDFs.

---

## Features

- Upload PDFs and extract text automatically.
- Split content into manageable chunks for better semantic search.
- Create embeddings and a vector store to enable fast, accurate retrieval.
- Answer user queries with detailed responses based on the PDF content.
- Interactive and easy-to-use UI built with **Streamlit**.

---

## Tech Stack

- **Python**
- **LangChain**
- **PyPDF2**
- **Chroma** (Vector Store)
- **Groq LLM**
- **Streamlit**
